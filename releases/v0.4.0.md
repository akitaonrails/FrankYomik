# v0.4.0

## Rendering improvements

- Improved furigana vertical rendering: bigger fonts, better column breaks, rotated long vowel mark (ー)
- Fixed text clipping in oval bubbles with conservative mask-safe bounding box
- Fixed font clipping and consistency, improved translation prompt
- Use bold font and full detection bbox for English text rendering

## Security hardening

- Hardened cache paths and WebView fetch boundaries
- Site matching in client now uses parsed host instead of regex over full URL
- Cache path components consistently sanitized (chapter, page rejected if unsafe)

## Infrastructure

- Fixed API cache permission denied in Docker
- Fixed CI: use full venv paths instead of source activate
- Added missing NaverWebtoonStrategy import
